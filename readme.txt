== Description ==

This simple plugin allows the WordPress site administrator to enforce minimal password requirements on its user. 

== Installation ==
1. Drop the 'obe-rules' folder in your plugin directory.
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Set your desired password requirements in Settings/oberules

WordPress hashes passwords before storing them in its database.  


1. Settings page.
